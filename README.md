# 🎬 Netflix Clone (React.js, TMDb API, Firebase)

A sleek and responsive **Netflix UI clone** built using **React.js**, powered by **TMDb API** for dynamic movie listings and **Firebase** for database and hosting functionality.

---

## 🚀 Features

- 🖼️ Real-time movie listings via **TMDb API**
- 📁 Category-wise movie browsing (Trending, Top Rated, Action, Comedy, etc.)
- 🔍 Movie details on hover
- ☁️ Firebase integration for database and hosting
- 🌐 Fully responsive UI

---

## 🧰 Tech Stack

- React.js (Hooks & Functional Components)
- TMDb API (The Movie Database)
- Firebase (Database, Hosting)
- Axios for API requests
- CSS / styled-components

---
